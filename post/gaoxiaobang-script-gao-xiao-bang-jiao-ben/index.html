【原创】Gaoxiaobang-script 高校邦脚本

2021-05-15 Javascript

高校邦刷课脚本运行在 Tampermonkey

主要功能：

  • 可自动点击播放.
  • 讨论题自动复制已有的答案并粘贴提交,若不需要请在代码中按提示删除(注释).
  • 自动跳过 "作业" "讨论".

你有两种使用方法:

  • 复制 GitHub Gxb.js全部代码，并粘贴至Tampermonkey(油猴)后即可使用．
  • 在Greasy Fork中安装本插件．
