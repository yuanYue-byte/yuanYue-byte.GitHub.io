【原创】环境配置PyCharm+Pyside+QtDesigner+PyUic+PyRcc

2021-06-08 Python

PyCharm下如何高效使用Pyside库开发GUI界面

  • 1.下载安装PySide库
    Ctrl+Alt+S进入设置界面 选择Python Interpreter 点击＋号

    搜索Pyside并下载 这里我选择Pyside6
  • 2.配置QtDesigner
    在设置界面选择External Tools点击+号 Program选择pyside6-designer.exe
    一般情况下路径为 工作目录/venv/Scripts/

  • 3.配置PyUic
    Program选择pyside6-uic.exe
    Argument填入 $FileName$ -o $FileNameWithoutExtension$.py
  • 4.配置PyRcc
    Program选择 pyside6-rcc.exe
    Argument填入$FileName$ -o $FileNameWithoutExtension$rc.py

注意Working directory应选择你的工作目录而不是工具所在目录
